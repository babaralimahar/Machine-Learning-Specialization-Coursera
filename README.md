## About This Specialization

The **Machine Learning Specialization** is a foundational program developed by **Andrew Ng** in collaboration with **DeepLearning.AI** and hosted on **Coursera**. It provides a broad introduction to modern machine learning, data science, and deep learning techniques.

This repository contains all programming assignments, optional labs, and practice materials completed across the three courses of the specialization. The code demonstrates practical implementations of machine learning algorithms using **Python**, **NumPy**, **TensorFlow**, and **scikit-learn**.

> "Machine learning is the science of getting computers to learn without being explicitly programmed."
> — **Andrew Ng**

</summary>

<br/>

<table>
<tr>
<td width="50%" valign="top">

**Week 1 — Introduction to Machine Learning**
- Overview of machine learning and its applications
- Supervised vs. unsupervised learning
- Univariate linear regression
- Cost function and gradient descent

**Week 2 — Regression with Multiple Input Variables**
- Multiple linear regression
- Vectorization with NumPy
- Feature scaling and normalization
- Feature engineering and polynomial regression

</td>
<td width="50%" valign="top">

**Week 3 — Classification**
- Binary classification and logistic regression
- Decision boundary
- Cost function for logistic regression
- Overfitting, regularization (L1 / L2)
- Regularized logistic regression

</td>
</tr>
</table>

**Key Libraries:** `NumPy` `Matplotlib` `scikit-learn`

</details>

<table>
<tr>
<td width="50%" valign="top">

**Week 1 — Neural Networks**
- Neurons, layers, and forward propagation
- TensorFlow and Keras implementation
- Activation functions: ReLU, Sigmoid, Linear

**Week 2 — Neural Network Training**
- Model compilation and training in TensorFlow
- Activation function selection
- Multiclass classification with Softmax
- Adam optimizer and learning rate

</td>
<td width="50%" valign="top">

**Week 3 — Advice for Applying Machine Learning**
- Bias-variance tradeoff
- Cross-validation and test sets
- Learning curves and error analysis
- Data augmentation

**Week 4 — Decision Trees**
- Decision tree learning and splitting
- Information gain and entropy
- Random forests and boosting (XGBoost)
- When to use trees vs. neural networks

</td>
</tr>
</table>

**Key Libraries:** `TensorFlow` `Keras` `NumPy` `scikit-learn`
