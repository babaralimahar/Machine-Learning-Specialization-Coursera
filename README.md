# Machine Learning Specialization – Coursera

This repository contains my **personal implementations, projects, and notes** from the  
**Machine Learning Specialization by Andrew Ng (Coursera)**.

The goal of this repository is to demonstrate my understanding of **core machine learning concepts**, algorithms, and practical implementation using Python.

---

## 📌 About the Specialization

The Machine Learning Specialization is a comprehensive program that covers:
- Fundamental machine learning concepts
- Practical algorithm implementation
- Model evaluation and improvement techniques
- Real-world ML problem-solving approaches

---

## 📚 Courses Covered

### 1️⃣ Supervised Machine Learning
- Linear Regression
- Gradient Descent
- Logistic Regression
- Regularization (L1, L2)
- Model Evaluation

### 2️⃣ Advanced Learning Algorithms
- Neural Networks
- Backpropagation
- Bias vs Variance
- Hyperparameter Tuning
- Model Optimization

### 3️⃣ Unsupervised Learning & Recommenders
- K-Means Clustering
- Principal Component Analysis (PCA)
- Anomaly Detection
- Recommendation Systems (Collaborative Filtering)
